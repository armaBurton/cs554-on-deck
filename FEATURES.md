    - end

- **List Generation**
  - Randomize/Lock at a set time

 # 3 User Stories
**AUTHENTICATION** As a *USER* I would like to use my Gmail account to quickly and securely create a user profile
**PROFILES** As a *USER* I would like to save my *NAME* and *PERFORMER NAME* in my account details so that I can sign up quickly for future events
**EVENT CREATION** As a *EVENT CREATOR* I would like to be able to add the *Event Details(Where, What, When)* when creating a new event
**LIST GENERATION** As a *EVENT CREATOR* I would like the list to randomize and/or lock at a set time prior to the start of the event to prevent modification
**LIST GENERATION** As a *EVENT CREATOR* I would like to be able to *add/remove/rearrainge* performers from the list after it has been locked
**LIST GENERATION** As a *PERFORMER* if I arrive after the list has been locked I would like to be able to add myself to the end of the list
